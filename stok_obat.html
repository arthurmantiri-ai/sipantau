    Import Data Stok dari Excel

    ×

    Gunakan fitur ini untuk memasukkan banyak data stok sekaligus tanpa input satu per satu. Langkahnya:

    1. Unduh template Excel di bawah ini.
    2. Isi data obat sesuai kolom pada template (jangan ubah nama kolom).
    3. Upload file yang sudah diisi, periksa pratinjau, lalu klik Proses Import.
